01 — Methodology
End-to-end pipeline
01
Data loading & EDA
Loaded the Telco Customer Churn dataset (7,043 rows, 21 columns) from Kaggle. Explored distributions, missing values, and found ~26% churn rate — indicating class imbalance.
pandas seaborn matplotlib
02
Data cleaning & preprocessing
Fixed TotalCharges column (spaces stored as NaN), dropped 11 null rows, removed customerID, encoded all categorical features with LabelEncoder, and scaled with StandardScaler.
LabelEncoder StandardScaler to_numeric dropna
03
Class imbalance — SMOTE
Applied SMOTE (Synthetic Minority Oversampling Technique) on training data only. Balanced 4,134 vs 1,491 samples to 4,134 vs 4,134, ensuring the model learns both churn and non-churn patterns equally.
SMOTE imbalanced-learn train_test_split
04
Model training & comparison
Trained three models — Logistic Regression (baseline), Random Forest, and XGBoost. Evaluated each on accuracy, precision, recall, F1-score, and ROC-AUC across 1,407 test samples.
XGBoost Random Forest Logistic Regression
05
Evaluation & visualization
Generated confusion matrix, ROC curve (AUC=0.80), and feature importance chart. Identified tenure as the strongest predictor — customers with shorter tenure churn significantly more.

smote_balancing.py
from imblearn.over_sampling import SMOTE
from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(
    X_scaled, y,
    test_size=0.2,
    random_state=42
)

smote = SMOTE(random_state=42)
X_bal, y_bal = smote.fit_resample(X_train, y_train)

model_training.py
from xgboost import XGBClassifier
from sklearn.metrics import roc_auc_score

model = XGBClassifier(
    eval_metric='logloss',
    random_state=42
)

model.fit(X_bal, y_bal)
y_pred = model.predict(X_test)
y_proba = model.predict_proba(X_test)[:,1]

auc = roc_auc_score(y_test, y_proba)
# ROC-AUC: 0.80
preprocessing.py
# Fix TotalCharges (spaces → NaN)
df['TotalCharges'] = pd.to_numeric(
    df['TotalCharges'],
    errors='coerce'

# Encode target safely
df['Churn'] = df['Churn'].str.strip()
df['Churn'] = df['Churn'].map({'Yes':1,'No':0})
df['Churn'] = df['Churn'].astype(int)
